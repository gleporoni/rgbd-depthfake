# rgbd-depthfake

Instruction to run the code:

1. Run the Docker Image.
```
./run.sh
```

2. Edit the ```config``` files accordingly to the experiment you want to execution. 

3. Run the ```train``` or ```test```.
```
python src/$select_a_file$.py
```
